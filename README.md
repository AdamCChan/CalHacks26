# CalHacks26
Repository for Timecapsule - a CalHacks26 project.

# Authors: Adam Chan, Lucas Kasdorf, Benny Liang


# Steps to run
1. cd in to `frontend` directory
2. run `npm i` (Only run on initial configuration of directory)
3. run `npm run dev`

new terminal:
1. cd into `backend` directory
2. run `npm i` (Only run on initial configuration of directory)
3. run `npm run dev`
